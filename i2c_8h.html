Detailed Description

I2C driver API for ARM Cortex-M (portable CMSIS-style).

This header defines the API, enums, and data structures for the I2C driver. It works with a hardware abstraction layer (hal_i2c.h) to remain portable across different ARM Cortex-M microcontrollers.
