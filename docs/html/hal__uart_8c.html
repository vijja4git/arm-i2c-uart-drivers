Detailed Description

Simulated Hardware Abstraction Layer for UART.

This HAL allows the UART driver to run on any host machine (Mac/Linux/Windows) without real hardware. It simulates:

  • UART TX ready state
  • UART RX ready state
  • DATA register behavior
  • Baudrate, stop-bit, parity configuration (stored but not functional)

For real embedded systems, replace the simulated registers with actual MCU register accesses.
