Detailed Description

Hardware Abstraction Layer for UART (Simulated CMSIS-style).

This HAL provides a hardware-agnostic interface for the UART driver. It simulates UART register operations so the project can compile and run on any platform (including host PC for testing).

On real hardware, replace the register definitions with actual MCU registers.
