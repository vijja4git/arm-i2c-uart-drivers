Detailed Description

Hardware Abstraction Layer for I2C (Simulated CMSIS-style).

This HAL provides a hardware-agnostic interface for the I2C driver. It simulates I2C register operations so the project can compile and run on any platform (including host PC for testing).

On real hardware, replace the register definitions and functions with actual MCU register accesses (e.g., STM32, NXP, TI).
