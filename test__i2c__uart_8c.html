Detailed Description

Unit tests for UART + I2C driver simulation.

These tests run on a host PC (no hardware required). They validate the behavior of simulated HAL layers, ensuring correct functionality of the firmware drivers.

This is not a full testing framework, but simple, clear tests suitable for demonstration in a portfolio.
