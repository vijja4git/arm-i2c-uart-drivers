Detailed Description

UART driver implementation for ARM Cortex‑M (portable CMSIS‑style).

This file provides a minimal, production‑style UART driver using C11. It is hardware‑independent and relies on the abstracted HAL layer defined in hal_uart.h. The goal is to demonstrate how low‑level embedded drivers are structured in real firmware projects.
